# Changelog

All notable changes to HydroDL2 will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.2.0] - 2026-02-XX

### Added
- Initial PyPI release
- HBV model variants: `hbv`, `hbv_1_1p`, `hbv_2`, `hbv_2_hourly`, `hbv_2_mts`, `hbv_adj`
- Data assimilation module: variational precipitation DA
- Public API: `load_model()`, `available_models()`, `available_modules()`
- License agreement prompt on first import
- CI/CD pipelines for testing, linting, and wheel building
